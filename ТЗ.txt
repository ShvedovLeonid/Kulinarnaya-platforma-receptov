ТЗ
Пользователи
Регистрация / логин / выход.
Личный кабинет – только свои рецепты.
Рецепты
Добавление, редактирование, удаление (только автором).
Поля: название, описание, ингредиенты, время приготовления, фото (загрузка файла).
Главная страница – карточки всех рецептов (фото, название, краткое описание, автор).
REST API
/api/recipes → список (id, название, время, автор, ссылка на фото).
/api/recipe/<id> → полные данные рецепта.
Технологии
Bootstrap для интерфейса.
SQLite + SQLAlchemy ORM.
requirements.txt.
Проект готов к хостингу (PythonAnywhere и др.).
